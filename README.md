# Professional Portfolio

This is a personal portfolio website for Saheedha Sumaya, showcasing AI, data science, and web development experience.

## Contents

- `index.html` — Home page with highlights, CGPA, work summary, core focus, technology stack, and project snapshot.
- `about.html` — About page with work experience, education timeline, and capabilities.
- `achievements.html` — Achievements page with professional highlights, major project achievements, and certification links.
- `contact.html` — Contact page with direct contact details and preferred discussion areas.
- `styles.css` — Global stylesheet for layout and appearance.
- `script.js` — Client-side script for interactive behavior.

## Notes

- This repository contains only static frontend files and does not require a build step.
- Open `index.html` in a browser to preview the website.
